自在生活

April 10, 2018   tools

Git

标签

列出已有的标签

git tag

模糊查询 tag

git tag -l ‘tagName*‘

附注标签

git tag -a <tagName> -m ‘comment’

显示指定标签的信息

git show <tagName>

过去的提交打标签

git tag -a v1.2 9fceb02 // 末尾指定提交的校验和（或部分校验和)

推送标签到远程

git push origin <tagName> // 单个 git push origin –tags // 所有本地

删除本地标签

git tag -d <tagName>

删除远程标签

git push origin :refs/tags/<tagName>

根据特定标签创建新分支

git checkout -b <branchName> <tagName>

历史

查看提交历史

git log –pretty=oneline

分支

分支创建

git branch testing

删除远程分支

git push origin –delete <branchName>

合并分支

git merge –no-ff -m “comment” <branchName>

更新远程分支列表

git remote update origin –prune

在本地创建分支dev并切换到该分支

git checkout -b dev origin/dev
